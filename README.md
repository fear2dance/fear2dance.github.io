## Publish on GitHub Pages

```bash
cd ~/Downloads
unzip kuanysh-faang-final-portfolio.zip
cd kuanysh-faang-final

git init
git add .
git commit -m "Launch professional portfolio"
git branch -M main
git remote add origin https://github.com/fear2dance/portfolio.git
git push -u origin main
```

Then open GitHub:

```text
Repository → Settings → Pages → Deploy from branch → main → /root → Save
```

Your site will be available at:

```text
https://fear2dance.github.io/portfolio/
```
