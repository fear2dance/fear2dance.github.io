Case study

JobTrust DWH

Verified Job Vacancy Intelligence Platform built as a production-style data warehouse project.

GitHub repositoryBack to portfolio
PythonPostgreSQLAirflowdbtDockerStreamlit

Problem

Job seekers need a clearer way to understand whether public job postings are fresh, active, and trustworthy. The project converts scattered job data into structured analytics.

Architecture

  • Public job sources feed Python ingestion workflows.
  • PostgreSQL stores normalized and analytical layers.
  • dbt builds repeatable transformations and warehouse models.
  • Airflow orchestrates the pipeline for production-style scheduling.
  • Streamlit presents freshness, trust scoring, and anomaly signals.

Recruiter signal

This project demonstrates pipeline design, data modeling, orchestration, Dockerized local development, and dashboard-oriented delivery.
