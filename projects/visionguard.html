Case study

VisionGuard AI

Computer vision project for AI-generated image detection with a product-style interface and real-time inference workflow.

GitHub repositoryBack to portfolio
PyTorchResNetDockerStreamlitCI/CD

Problem

AI-generated images are increasingly difficult to evaluate manually. VisionGuard creates a structured image-authenticity workflow with prediction output and confidence scoring.

Architecture

  • Image upload and preprocessing workflow.
  • ResNet-based classifier implemented in PyTorch.
  • Inference layer returns prediction and probability score.
  • Streamlit interface presents the result in a clear, reviewer-friendly format.
  • Docker and CI/CD structure support reproducible deployment.

Recruiter signal

This project demonstrates applied ML engineering, app packaging, model inference, UX thinking, and deployment-aware project structure.
